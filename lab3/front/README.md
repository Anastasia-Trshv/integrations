# Books & Authors Management

React приложение для управления книгами и авторами с интеграцией REST API.

## Функциональность

✅ **CRUD операции для Authors**
- Получение списка всех авторов
- Получение автора по ID
- Создание нового автора
- Обновление автора
- Удаление автора

✅ **CRUD операции для Books**
- Получение списка всех книг
- Получение книги по ID
- Создание новой книги
- Обновление книги
- Удаление книги

✅ **Идемпотентность**
- Поддержка `Idempotency-Key` заголовка для POST операций
- Возможность включить/выключить идемпотентность для создания записей
- Предотвращение дублирования при повторных запросах

✅ **Обработка ошибок**
- Централизованная обработка ошибок через axios interceptors
- Понятные сообщения об ошибках для пользователя
- Обработка сетевых ошибок и ошибок сервера

✅ **UI/UX**
- Современный и отзывчивый интерфейс
- Формы для создания и редактирования
- Модальные окна
- Индикаторы загрузки
- Обработка состояний ошибок

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` в корне проекта (опционально):
```env
VITE_API_BASE_URL=http://localhost:5042
```

3. Запустите dev сервер:
```bash
npm run dev
```

4. Откройте браузер по адресу, указанному в консоли (обычно `http://localhost:5173`)

## Использование идемпотентности

Для создания записей с идемпотентностью:

1. На странице Authors или Books нажмите кнопку "Включить идемпотентность"
2. Создайте новую запись - будет использован тот же `Idempotency-Key`
3. Если отправить запрос повторно с тем же ключом, сервер вернет существующую запись без создания дубликата

## API Endpoints

### Authors
- `GET /api/v1/Authors/GetAllAuthors` - Получить всех авторов
- `GET /api/v1/Authors/GetAuthorById/{id}` - Получить автора по ID
- `POST /api/v1/Authors/CreateAuthor` - Создать автора (с Idempotency-Key)
- `PUT /api/v1/Authors/UpdateAuthorById/{id}` - Обновить автора
- `DELETE /api/v1/Authors/DeleteAuthorById/{id}` - Удалить автора

### Books
- `GET /api/v1/Books/GetAllBooks` - Получить все книги
- `GET /api/v1/Books/GetBookById/{id}` - Получить книгу по ID
- `POST /api/v1/Books/CreateBook` - Создать книгу (с Idempotency-Key)
- `PUT /api/v1/Books/UpdateBookById/{id}` - Обновить книгу
- `DELETE /api/v1/Books/DeleteBookById/{id}` - Удалить книгу
